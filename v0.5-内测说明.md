# 夜聊 Lite v0.5 内测说明

感谢你帮忙试用夜聊 Lite。

这是一款本地优先的轻量 AI 聊天页面。API Key 只保存在你自己的浏览器里，不会写进页面文件，也不会出现在聊天备份、诊断信息或反馈模板中。

## 本次重点测试

- 能否正常填写并保存 DeepSeek / 通义千问 API Key。
- 设置里的“测试当前模型连接”是否能判断接口可用。
- 底部模型切换是否顺手。
- 不同会话是否能记住各自使用的模型。
- 发送消息、流式回复、复制、删除、重新生成是否正常。
- 会话新建、切换、搜索、重命名、删除是否正常。
- JSON 备份导入/导出是否正常，且不会包含 API Key。
- “复制诊断信息”和“复制反馈模板”是否方便反馈问题。
- 手机端建议优先用 Safari 打开，电脑端建议优先用 Chrome。

## v0.5 新增内容

- 新增 DeepSeek / 通义千问模型预设，并保留自定义 OpenAI 兼容接口。
- 新增底部模型切换，已保存 Key 的模型可以快速切换。
- 新增会话级模型记忆，每个会话会记住上次使用的服务商和模型。
- 新增模型连接测试，填好 Key 后可以先确认接口是否可用。
- 新增诊断信息复制，方便定位浏览器、模型、参数和最近错误。
- 新增内测反馈模板，遇到问题时可直接复制粘贴给维护者。
- 新增内测发布小面板，用来检查版本、发布包和安全提醒。

## 已知限制

- 联网轻量搜索仍是实验功能，在国内网络环境下可能不稳定。
- 对话记录保存在当前浏览器 localStorage，换浏览器需要导出/导入 JSON 备份。
- 这是单文件静态页面，没有账号系统和云同步。
- API Key 一旦填入当前浏览器，本机使用者都可能使用该 Key，请不要在不可信设备上保存。
- MCP / 工具服务接口仍是预留能力，尚未做完整工具调用流程。

## 反馈方式

如果遇到问题，请打开“设置”，点击“复制反馈模板”，把复制出来的内容发给维护者即可。

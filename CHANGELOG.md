# 更新记录

## v0.5 - 内测候选版

- 支持 DeepSeek 和通义千问两个模型预设。
- 支持自定义 OpenAI 兼容接口。
- 支持底部快速切换模型服务商。
- 支持每个会话记住自己的模型配置。
- 支持设置里测试当前模型连接。
- 支持复制诊断信息，且不包含 API Key 或聊天内容。
- 支持复制内测反馈模板。
- 支持设置里的内测发布小面板。
- 优化“删除本地 API Key”，会清理当前模型对应的本地 Key 记录。
- 发布包 README 已补充内测、安全和浏览器建议。

## v0.4 及更早

- 支持 DeepSeek 流式对话。
- 支持本地 localStorage 保存会话。
- 支持多会话管理、会话搜索、重命名和删除。
- 支持消息复制、删除、重新生成。
- 支持 Markdown 渲染、代码高亮、代码块复制。
- 支持完整 HTML 代码块 iframe 预览。
- 支持角色模式、快捷提示词、参数预设和上下文长度控制。
- 支持当前对话 Markdown 导出。
- 支持全部会话 JSON 备份导入/导出，且备份不包含 API Key。
- 支持深色/浅色主题。
- 支持联网轻量搜索实验功能。
